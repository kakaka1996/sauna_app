サービス概要
サウナ施設からサウナ体験した後のサウナ飯まで一気貫徹に調べられるアプリ 自分のサウナ体験を記録し、自分にとってどのようなサウナがベストなのかを追跡できる

このサービスへの思い・作りたい理由
２年前くらいに友人からサウナに誘われるようになり、今では毎月欠かさず行くほどまでになりました。既存のサウナ施設検索アプリを使って気になるサウナ施設に足を運んでいましたが仕事終わりに行き、閉店ギリギリの23時24時近くまで営業している施設から帰るときに近くに営業しているごはん屋さんが少なく検索しても営業してかつ行きたいと思えるようなお店がなかなか出てこないことがありました。 そこであらかじめ自分の行きたいと思えるサウナ施設とサウナ後に食べれる店が一緒に検索できるアプリができればいいなと思い、アプリ案としました。

ユーザー層について
サウナ後のごはん探しに迷うサウナユーザー。サウナに興味があるが、どこの施設がいいかわからない人

サービスの利用イメージ
すでにサウナが趣味の人はどこにサウナ施設がありサウナ飯がどこにあるのか一括で管理でき、自分に対してどのような時間配分が満足度が高いのか可視化することができる。 まだ、体験したことのない人に対しては興味を持ってもらえるきっかけになる

ユーザーの獲得について
 →SNSでの広告、サウナに行く友人に使用してもらう

サービスの差別化ポイント・推しポイント
従来あるアプリではサウナ施設の検索や記録のみだったり、サウナ施設と周囲のサウナ飯の検索機能だけであったが、１つのアプリでサウナ施設の検索、サウナ体験の記録からサウナ飯の検索まで完結できるアプリとした。
